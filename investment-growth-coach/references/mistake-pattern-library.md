# Mistake Pattern Library

Use periodically to summarize recurring errors from decision reviews.

```text
Pattern name:
Typical trigger:
Past examples:
Damage:
Early warning phrase:
Coach reminder:
Replacement rule:
```

Examples:

- First-big-down-day dip buying.
- Chasing after consecutive rises.
- Adding to prove a thesis.
- Full switching after seeing another theme rise.
- Ignoring counter-evidence.
- Over-concentrating in related funds.

When a similar situation appears, remind the user of the matching pattern before action.
