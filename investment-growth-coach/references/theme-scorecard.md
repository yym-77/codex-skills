# Theme Scorecard

Use this to judge attention priority, not automatic buying.

```text
Theme:
Policy support: __/5, evidence:
Industry trend: __/5, evidence:
Earnings/order validation: __/5, evidence:
Capital strength: __/5, evidence:
Valuation/position: __/5, evidence:
Current stage: theme / watch / small trial / confirmed main theme / rising / crowded / divergence / fading / repair
Counter-evidence:
1.
2.
3.
Invalidation conditions:
If invalidated, action:
Learning output:
```

Rules:

- High scores do not mean buy. They mean "study first."
- If position is overheated or user is over-concentrated, action can still be observe, pause, or reduce.
- Always include "if invalidated, action."
