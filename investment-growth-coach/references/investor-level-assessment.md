# Investor Level Assessment

Use this self-test when the user starts the skill, asks for a growth plan, wants to upgrade tools, or shows repeated confusion.

Ask concise questions and infer level from evidence. Do not require all questions every time.

## Core Questions

1. Do you know the difference between fund estimate and final net value?
2. Do you know the 15:00 rule for off-exchange fund subscriptions/redemptions?
3. Can you state total investable capital, invested amount, remaining cash, and emergency cash?
4. Do you have a single-theme position limit?
5. Have you reviewed decisions for at least 4 consecutive weeks?
6. Can you explain what your current funds mainly hold?
7. Do you know maximum drawdown?
8. Can you accept a short-term 10%-20% loss without panic selling?
9. Can you write "what proves me wrong" before buying?
10. Do you know the difference between off-exchange funds and exchange-traded ETFs?
11. Can you read a fund quarterly report and top holdings?
12. Can you read basic company financial statements?

## Level Assignment

- L0 if the user lacks basic fund/cash/risk concepts.
- L1 if the user understands basic fund rules but lacks review discipline.
- L2 if the user can review and explain simple theme strength/weakness.
- L3 if the user understands ETF mechanics and can write a trade plan.
- L4 if the user can map industries and connect evidence to earnings.
- L5 if the user can complete stock research in simulation.
- L6 if the user has small-stock practice with strict sizing and review.

When uncertain, assign the lower level.
