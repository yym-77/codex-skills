# Cognitive Bias Checklist

Check these before any operation advice:

- FOMO: buying because it rose and the user feels left behind.
- Panic: selling because it fell and the user wants relief.
- Anchoring: focusing on cost price instead of current evidence.
- Confirmation bias: only seeking arguments that support the user's view.
- Loss aversion: adding or selling mainly to reduce emotional pain.
- Sunk cost: adding to prove the original decision was right.
- Herding: following a hot topic because others discuss it.
- Overconfidence: ignoring risk after several correct calls.
- Recency bias: treating today's move as the whole trend.
