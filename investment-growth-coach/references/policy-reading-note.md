# Policy Reading Note

Use policy as a watch-pool signal, not an automatic buying reason.

```text

Issuing body:
Release date:
Main content:
Related industries:
Possible beneficiaries:
What real implementation needs: funding / projects / orders / standards / production / earnings
Why this is not an immediate buy signal:

```

Prefer official policy documents and reputable news. Label unverified summaries.
